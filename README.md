# Análisis de Factores que Predicen el Rendimiento Académico

**Autor:** José Alejandro Mora Poblete  
**Curso:** Analítica de Datos Industriales (ADI&TD)  
**Programa:** Doctorado Interinstitucional en Ingeniería Industrial – Di3  

## Descripción

Este trabajo aplica un modelo de regresión lineal múltiple para predecir 
el puntaje de examen de estudiantes a partir de factores como horas de 
estudio, asistencia, motivación y contexto familiar, entre otros.

## Dataset

- **Fuente:** Kaggle – Student Exam Performance Dataset  
- **Link:** https://www.kaggle.com/datasets/grandmaster07/student-exam-performance-dataset-analysis  
- **Observaciones:** 6.607 estudiantes  
- **Variables:** 20
